# Rishi Vidyalaya Digital Presence

Welcome to the digital home of Rishi Vidyalaya — a polished, modern, and user-first website built to showcase the school’s mission, academic strengths, campus experiences, and contact pathways.

This project delivers:
- A fresh, responsive website experience
- Rich content for admissions, labs, events, and student development
- Smooth navigation and modern UI interactions
- Built with fast, scalable frontend tooling

---

## ✨ Key Highlights

- **Modern React + Vite app**
- **Responsive design** for desktop and mobile
- **Animated and interactive sections**
- **Accessible navigation**
- **Sharable pages** for academics, events, labs, admissions, and more

---

## 🌟 What’s Included

This website is structured into meaningful pages, including:

- Home / Index
- About
- Academics
- Admissions
- Events
- Contact
- Labs like AI Robotics, Automotive, and more
- Digital Fest, Field Trips, Project-Based Learning, Holistic Development, and additional school experiences

---

## 🛠️ Built With

- **React 18**
- **Vite**
- **TypeScript**
- **Tailwind CSS**
- **Radix UI / shadcn UI**
- **Framer Motion**
- **React Router**
- **React Query**
- **Recharts**
- **Playwright & Vitest** for testing
